RhythmMad

Descripción

Este proyecto representa una aplicación móvil utilizando React Native, diseñada para proporcionar información actualizada sobre eventos en la ciudad de Madrid. Su objetivo principal es facilitar la búsqueda y visualización de eventos como conciertos, exposiciones, conferencias y actividades culturales, brindando una experiencia fluida e intuitiva a los usuarios.

Además de funcionalidades básicas de navegación y visualización de datos, la aplicación incorpora características avanzadas como la posibilidad de agregar nuevos eventos a la base de datos Firebase, así como acceder a información detallada de cada evento.
